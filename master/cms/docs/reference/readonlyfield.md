# ReadonlyField
Read-only field, created from label and span tags

{{:readonlyfield.png|:readonlyfield.png}}

### Example use

~~~ {php}
new ReadonlyField(
   $name = "readonly",
   $title = "Title",
   $value = "SilverStripe open source CMS"
)
~~~
