====== HiddenField ======
Hidden field.





==== Example use ====
If you want to implement a [[HiddenField]] into a form element, you need to pass it an array of source data.

<code php>
new HiddenField (
   $name = "hidden",
   [$title = "title"],
   [$value = "value"]
)
</code>
