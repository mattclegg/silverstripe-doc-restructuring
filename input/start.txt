====== Welcome to the SilverStripe developer documentation wiki ======

This wiki documents high-level features of the [[http://www.silverstripe.org|SilverStripe open source platform]], aimed at developers. 
If you would like to contribute, you can [[http://doc.silverstripe.com/doku.php?id=start&do=register|register]] and then [[http://doc.silverstripe.com/doku.php?do=login&id=start|login]].
==== Getting support ===
  * Use our [[http://silverstripe.org/forums/|support forum]] and [[http://silverstripe.org/irc-channel|irc channel]]
  * See [[http://userhelp.silverstripe.com|userhelp]] for end-user documentation
  * Visit some of the [[community run third party websites]]
