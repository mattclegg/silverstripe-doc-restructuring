====== LanguageDropDownField ======
An extension to dropdown field, pre-configured to list languages.

{{:languagedropdownfield.png|:languagedropdownfield.png}}

===== Code example =====
<code php>
new LanguageDropdownField(
    $name='lang',
    $title = 'Please choose your language'
)
</code>
