===== Introduction =====
Youtubeservice makes your life easy when it comes to online video content by providing easy integration with popular video service - [[http://www.youtube.com|YouTube]]. Youtubeservice exposes you the GData API of YouTube, allowing you to create mashups of video content. There are two ready made components under this module. Those are :
  * YoutubeGallery page type -  create a video galleries based on YouTube's videos.
  * YoutubeWidget - show your favorite videos on your blog's sidebar

===== Requirements =====
You need to use the version **2.2.0** of SilverStripe.

===== Module Installation =====
  - Download the [[http://www.silverstripe.com/assets/downloads/modules/youtubeservice-0.1.tar.gz|Youtubeservice]] module
  - After extraction, place this directory in the root of your SilverStripe installation
  - Run db/build
  - The CMS should now have a "YoutubeGallery" page type
  - If you have blog module installed, you'll get YoutubeWidget for blog module.
