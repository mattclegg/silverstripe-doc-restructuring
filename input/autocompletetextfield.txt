====== AutocompleteTextField ======
Autocompleting text field, using script.aculo.us
 
===== Example use =====

<code php>
new AutocompleteTextField(
   $name = "song",
   $title = "Please enter your favourite song name",
   $optionsURL = "http://yoursite.com/getyourvalues" 
)
</code>
