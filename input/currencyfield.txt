====== CurrencyField ======
Currency field.

{{:currencyfield.png|:currencyfield.png}}

Validation accepts only currency values

{{:currencyfieldvalidation2.png|:currencyfieldvalidation2.png}}

===== Example use =====

<code php>
new CurrencyField(
   $name = "price",
   $title = "Price",
   $value = "125.50"
);
</code>
