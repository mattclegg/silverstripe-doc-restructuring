====== CheckboxField ======
Single checkbox field.

{{:checkboxfield.png|:checkboxfield.png}}


==== Example use ====
First add the new field to the database:
<code php>
"db" => array(
   'newsletter' => 'Boolean'
)
</code>
Then add the field to the form
<code php>
new CheckboxField(
   $name = "newsletter",
   $title = "I would like to receive a monthly newsletter"
);
</code>
