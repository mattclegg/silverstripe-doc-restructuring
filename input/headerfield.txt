====== HeaderField ======
Field that generates a heading tag.
This can be used to add extra text in your forms.

{{:headerfield.png|:headerfield.png}}

==== Example use ====
If you want to implement a [[HeaderField]] into a form element, you need to pass it an array of source data.

<code php>
new HeaderField (
    $title = "header",
    $headingLevel = "2"
)
</code>
