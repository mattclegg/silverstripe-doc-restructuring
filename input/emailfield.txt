====== EmailField ======

Text field with email validation.

{{:emailfield2.png|:emailfield2.png}}

Validation accepts only valid e-mail addresses

{{:emailfieldvalidation.png|:emailfieldvalidation.png}}

==== Example use ====

<code php>
new EmailField(
   $name = "email",
   $title = "E-mail",
   $value = "joe.bloggs@gmail.com"
);
</code>
