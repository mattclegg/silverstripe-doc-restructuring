====== LabelField ======
Simple label tag. This can be used to add extra text in your forms.

{{:labelfield.png|:labelfield.png}}

==== Example use ====
If you want to implement a [[LabelField]] into a form element, you need to pass it an array of source data.

<code php>
new LabelField (
   $title = "A label"
)
</code>
