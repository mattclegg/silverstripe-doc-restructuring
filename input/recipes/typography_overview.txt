===== testing typography.css =====


If you are familiar with typography.css and how it is used within silverstripe (creating a WYSIWYG editor in the CMS - see http://doc.silverstripe.com/doku.php?id=typography) then the page below maybe useful for testing the typography as it applies to your pages. Just create a page called TypographyTestPage in your code and after running a http://www.yoursite.com/db/build/?flush=1 you create a TypographyTestPage to see your typography.css at work:
