====== Tools ======

If you're going to build sites with SilverStripe, you will need to make use of tools beyond the SilverStripe package itself.  Are number of tools that are known to work have been listed below.





===== IDEs and Editors =====

SilverStripe developers have had success working with the following tools for developing their PHP classes and .ss templates.  They range from simple "programmer's text editors" to full-blown IDEs.
